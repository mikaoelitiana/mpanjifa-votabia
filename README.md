# Kilia Votabia

Simple Electron UI client for [Votabia](https://github.com/tagip/votabia)

![Screenshot](./screenshot.png)

## Install

Run 

```
npm install
npm run dist-mac # For OSX
npm run dist-linux # For Windows
npm run dist-win # For Linux
```
